# Math Foundations for Machine Learning

This repository documents my structured study of the mathematical foundations required for professional-level Machine Learning and Data Science.

The goal is to move beyond library usage and develop a rigorous understanding of:

- Probability theory
- Multivariate distributions
- Bayesian inference
- Maximum Likelihood Estimation (MLE)
- Statistical inference
- Linear algebra for ML
- Optimization and gradient descent
- Regularization (L1/L2)
- Bias–variance tradeoff
- Information theory (entropy, KL divergence)
- Cross-validation and model evaluation

Each topic includes:
- Mathematical derivations
- Worked exercises
- Implementations from scratch in Python
- Comparisons with standard ML libraries
- Structured technical notes

This repository is part of my transition toward production-level Machine Learning engineering and advanced Data Science.

---

## Repository Structure

- `theory_notes/` → Structured conceptual summaries
- `derivations/` → Formal mathematical derivations
- `exercises/` → Solved probability and statistics exercises with formal derivations
- `python_implementations/` → Models implemented from scratch
- `notebooks/` → Weekly study notebooks
- `mini_project/` → Final integrative technical project
- `data/` → Datasets used
- `images/` → Visualizations and diagrams

## Objective

To build deep mathematical intuition and technical rigor required for designing, training, and deploying machine learning systems professionally.
